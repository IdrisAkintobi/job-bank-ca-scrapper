# jobbank.gc.ca scrapper

## Description

This is a simple Node.js application for scrapping jobs on jobbank.gc.ca.

### Requirements

-   [Node.js](https://nodejs.org/en/) v20.0.0 or higher

#### Installation

```bash
$ npm install
```

#### Running the app in development mode

```bash
$ npm run start:dev
```

#### Build the app

```bash
$ npm run build
```

#### Running the app

```bash
$ npm run start
```
